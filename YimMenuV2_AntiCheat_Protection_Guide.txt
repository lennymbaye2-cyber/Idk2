### Feature Shows "Warning: Anticheat bypass not detected"

**Solution 1: Install FSL**
- Download FSL from trusted source
- Place version.dll in GTA V directory
- Restart game

**Solution 2: Verify FSL Loaded**
- Check Windows Task Manager
- Look for "Rockstar Launcher" process
- Confirm "version.dll" is loaded
- Check Documents/GTAV Enhanced/Profiles exists

**Solution 3: Disable BattlEye**
- Rockstar Launcher → Settings
- Disable BattlEye
- Restart game

### Feature Executes But No Effect

**Possible Causes:**
- FSL not actually providing bypass (check logs)
- Game version mismatch
- Feature already running (disabled by rate limiter)
- Exception caught and silently failed

**Solution:**
1. Check notification carefully
2. Verify FSL status
3. Wait 3+ seconds between attempts (money drops)
4. Try again

### Game Crashes When Using Feature

**This Should Not Happen** - Exception handling prevents this

**If it does:**
1. Feature has bug
2. Game state is corrupted
3. Conflict with another mod

**Recovery:**
1. Load previous save
2. Disable problematic feature
3. Report issue with crash log

### Account Gets Flagged
