## TRANSACTION TYPES

1. BASKET Transactions
   - Multiple items in one transaction
   - Can add several items at once
   - Common for bulk purchases
   - Example: Buy multiple vehicle mods in one go

2. SERVICE Transactions
   - Single service action
   - Direct service purchase
   - Example: Unlock a specific feature

## SAFETY RECOMMENDATIONS

✅ Safe Transactions:
- BUY_VEHICLE with legitimate vehicles
- BUY_VEHICLE_MODS with standard mods
- BUY_PROPERTY with real properties
- BUY_CASINO_CHIPS (normal amounts)
- CREATE_PLAYER_APPEARANCE

❌ Risky Transactions:
- Avoid banned items (see list above)
- Don't use debug/exploit services
- Avoid extreme money amounts
- Don't use contraband items recklessly
- Avoid multiple rapid transactions (may flag AntiCheat)

⚠️ Bank Account Safety:
- Always use legitimate properties/vehicles
- Match prices to realistic values
- Don't spam transactions
- Use proper laundering methods if modifying money
- Consider using anticheat bypass if available


## TROUBLESHOOTING

Issue: "Failed to create basket"
- Solution: Make sure category and action are valid pairs
- Solution: Ensure shop_controller script is active

Issue: Transaction rejected by server
- Solution: May be flagged by anticheat
- Solution: Use anticheat bypass features
- Solution: Wait before attempting again

Issue: Items not appearing in inventory
- Solution: Check inventory category matches transaction
- Solution: Verify item hash is correct
- Solution: Check if item is currently available
